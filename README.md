# 行政期間ミニ

行政書士実務で使うことを想定した、期間計算アプリのHTMLプロトタイプです。

## できること

- 行政不服審査法・行政手続法を選択
- 手続・条文をプルダウンで選択
- 黄色マーカーの期間をタップ
- 今日・昨日・明日・特定日付から、満了日・参考締切を表示
- 「相当の期間」「標準処理期間」などは日数を手入力して計算
- iPhoneのホーム画面追加向けに、ゆるいカレンダーアイコン付き

## ファイル構成

すべて同じ階層に置いてあります。

- `index.html`
- `calendar-icon.svg`
- `calendar-icon.png`
- `manifest.webmanifest`
- `README.md`

`assets/` などの下層フォルダはありません。

## GitHub Pagesで公開する場合

1. GitHubで新しいリポジトリを作る
2. このフォルダ内のファイルをすべてアップロード
3. Settings → Pages → Branch を `main` / root に設定
4. 数分後、GitHub PagesのURLで `index.html` が開けます

## 注意

このプロトタイプは確認補助です。祝日、年末年始、閉庁日、個別法の特則、電子申請の受付時刻、実際の教示・通知内容は自動判定していません。実務では必ず e-Gov 法令検索、個別の通知・教示、行政庁の運用を確認してください。

参照元：

- e-Gov 法令検索：行政不服審査法（平成二十六年法律第六十八号）
- e-Gov 法令検索：行政手続法（平成五年法律第八十八号）
- e-Gov 法令検索：民法（明治二十九年法律第八十九号）第百三十八条〜第百四十三条
